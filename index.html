Microinteractions - IDM241

Click to Original Interface

Information

Trigger

  • Hover over the left or right arrow
  • Click on the left or right arrow

Rules

Frontend
  • Transparency increase to 80% when hover over
  • Transparency returns to 0% when not hovering
  • Next image fades in, while previous image fades out in a time frame of 1 second
Backend
  • The arrow clicking notifies a database that a specific image is selected and the database displays the proper event detail and "learn more" link

Feedback

  • Arrow box becomes transparent
  • Next image displays
  • The slide indication circle (bottom right) moves to the next position, indicating that this is the next slide
  • Image/event detail text changes based on image

Loops/mode

  • None
Loops and Modes

Images
