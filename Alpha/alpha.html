Information

Trigger

  • Hover over Slideshow

Rules

Frontend
  • Overlay color #000637 with Transparency from 75% to 90% transition-duration 3s.
  • Text font style Futura, Light Condense,color: #FFFFFF, font-size: 50, line-height: 60, letter-space: 5%, left-aligned, transitions from bottom left at 55% Transparency to center left at 100% over a duration of 3 seconds.
Backend
  • The arrow clicking notifies a database that a specific image is selected and the database displays the proper event detail and "learn more" link

Feedback

  • Arrow box becomes transparent
  • Next image displays
  • The slide indication circle (bottom right) moves to the next position, indicating that this is the next slide
  • Image/event detail text changes based on image

Loops/mode

  • None
